## What is CI/CD?

**CI/CD** stands for **Continuous Integration** and **Continuous Delivery/Deployment**. It is a set of practices and tools that automate the process of building, testing, and deploying software.

- Without CI/CD → developers write code, manually test it, then manually deploy it. This is slow, error-prone, and stressful.
- With CI/CD → every code change automatically goes through a pipeline that builds, tests, and deploys the software. It's fast, reliable, and consistent.

```
Developer pushes code
        ↓
  Code is built automatically
        ↓
  Tests run automatically
        ↓
  Code is deployed automatically
        ↓
  Users get the new version 🎉
```
